## Important Setup Instructions

⚠️ **Disclaimer**: Before running this project, please ensure the following:

### Required Configuration Files

- `.env` file must exist in the **server** folder
- `pdfhandler.json` file must exist in the **server** folder
- another file `.env` file must exist in the **client** folder (*server/.env* and *client/.env* are not the same)

### Running the Project

To run this project, you need to use **2 separate terminals**:

1. **Terminal 1** - CD to the `server` folder and execute:
```bash
   npm run dev
```

2. **Terminal 2** - CD to the `client` folder and execute:
```bash
   npm run dev
```

Both servers must be running simultaneously for the application to work properly.
